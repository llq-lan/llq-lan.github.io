SQL的 CRUD (增、删、改、查) 

创建一条数据 insert into [表名] (字段1，字段2……) values('字段值1','字段值2'……)
删除一条数据 delete from [表名]  where  id = 123  { 根据字段来删除 比如 id}
更新一条数据  update [表名] set (字段1='新的字段1值') where id = 123 {根据ID来更新一条数据}
查询数据 select * from [表名]  ==> 
查询一条数据 select [字段名]  from [表名] where [字段名] ='1234' 
模糊查询  select * from  [表名]  where [字段名] like '%【包含的字符】%'
